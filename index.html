Descriptive -> Prescriptive

Artifacts to enable descriptive to the prescriptive.

This page is a collection of artifacts created in order to explore the idea of moving from descriptive to a predictive environment.


AI playing Tic-Tac-Toe

Using self-play and a simple neural network to teach a simple AI agent how to play Tic Tac Toe. The main reason for this implementation, is not to build the fastest and best Agent to play human level Tic Tac Toe, but rather to provide an easy learning experience to anyone new to NN.

In order to understand the influence of problem representation on design. The simple game of Number Scrabble is also included as a link below. The same AI agent learned how to play the game.

Sentiment analysis

A sequential neural network was trained on 25 000 IMDB movie reviews. Based on the review, the model predicts the movie rating. It can be argued that this model can be used on any text and classifying if the text is positive or negative.

Note this model was built on a Sequential Neural Network using Keras. This approach has some limitations, this approach does not cater for the example: 'The movie Terrible, was excellent' vs 'The movie Excellent, was terrible'. ToDo: Investigate a recurrent neural network as an improvement.
