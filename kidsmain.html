Artifacts that can help kids learn.

How do humans and computers learn? We learn by either learning the theory (the rules) or by practising and learning from data. For example, computer can be given the rule: 2 x 7 = 14. Or it can practice and play, trying different answers to 2 x 7, until it realises 2 x 7 = 14. The first example where the rules are given, is "Rule Based" and the second example is "Data-Driven". Below is a practical set of artifacts that can be used to learn maths. Firstly the times table (rules) are provided and secondly the exercises (data-driven).
